HTML Structure - top

This theme have fixed layout with these partial elements:

- header with logo, pages and custom menu, header ad and categories menu,

- featured sliders,

- recent posts section (index),

- widget ready section (index),

- featured portfolio items (index),

- ad sections (index an single),

- featured posts section (index),

- newsletter and ads block (sidebar),

- widget ready section (sidebar),

- widget ready sections (footer),

- social networks, twitter, about us and contact form section (footer),

- portfolio template (3, 4 - columns),

Theme is available with custom pages templates: Fullwidth.

 

 

CSS Files and Structure - top

You can use five different color schemes (can be set in theme options):

- default (style-Default.css),

- light (style-Light.css),

- blue (style-Blue.css),

- brown (style-Brown.css),

- green (style-Green.css),

- mahagon (style-Mahagon.css).

All css styles are well formatted for easy reading / editing and are placed in main theme directory.

 

 

JavaScript - top

Mundo theme imports these Javascript files.

  1. lightbox.js (lightbox directory)
  2. coin-slider.min.js (js directory)
  3. jquery-1.2.6.min.js (js directory)
  4. sprinkle.js (js directory)
  5. opt_script.js (functions directory)
  6. jquery-ui-personalized-1.5.2.packed.js (js directory)
  7. jquery.tiptip.minified.js (js directory)
  8. hoverIntent.js (js directory)
  9. superfish.js (js directory)
  10. pngfix.js (js directory)
  11. jquery.tweet.js (js directory)

These javascript libraries enables slider effect, cool images preview etc.

 

 

Theme Setup - top

Theme Installation:
  1. Unzip the downloaded file,
  2. Upload Mundo folder into Themes folder in your WordPress installation (/wp-content/themes),
  3. Go to Appearance >Themes, find Mundo theme and activate it,
  4. Go to Dashboard > Mundo theme options and setup your theme.
Images Setup:
  1. Go to Dashboard > Settings > Media,
  2. Set Thumbnail size to 200x200px,
  3. Set Medium sze to 300x3000px
  4. Set Large sze to700x5000px

 

 

Theme Option page - top

Main Setup:
  1. Paste full URL of your logo image,
  2. Choose color scheme,
  3. Insert your Twitter account name,
  4. Exclude categories (if you want to),

Layout Setup:
  1. Turn on/off layout blocks to customize the look of your website.

Slideshow and Portfolio section:
  1. Choose category for Coin slider,
  2. Choose category for portfolio.

Featured sections:
  1. Choose category for featured section in sidebar,
  2. Enter embed video code (in 295x190px resolution).

 

About Us, Newsletter and Contact Settings:
  1. Fill following columns to setup About Us section,
  2. Enter e-mail adress,
  3. Setup Newsletter - Use your feedburner email subscriptions code

Social Networks Settings:
  1. Enter the full URL of social network or your social network profile or account,

Ads Setup:
  1. Insert URL of ad images and destination URLs to setup ads areas or paste AdSense code,
  2. Disable partial ad sections (if you want to),

 

 

 

If you are finished click on "Save changes" button!

 

 

 

Custom Templates, Thumbnails and Images - top

Custom Page Templates:

  1. Create a page by going to Page > Add New.
  2. In Attributes Section select custom page template (Sitemap, Full Width Page or Portfolio)
  3. Click the Publish button to publish your page.

Creating Portfolio Page

  1. Create page using Portfolio template (see previous steps). Done! But, do not forget to choose portfolio category in theme options menu.

Images and Thumbnails Setup:

  1. It is easy! Images and thumbnails are generating automatically. In Add or or Etit post screen upload and add image to your post content. And that is all!
